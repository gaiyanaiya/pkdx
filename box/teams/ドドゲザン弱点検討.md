- **ガブリアス** — じめん弱点STAB(じしん)、SPE102
- **ドリュウズ** — じめん弱点STAB(じしん)、SPE88。Cat D重複(はがね半減)

### ★★ 弱点技持ち（非STAB）/ STAB等倍+先手

- **バンギラス** — けたぐり(かくとう×4・非STAB)、可変威力
- **メガキラフロル** — だいちのちから(じめん×2・非STAB)
- **エルフーン / ロトム / ペリッパー / ミロカロス / ギャラドス / イダイトウ / メガフラエッテ / アシレーヌ / ニンフィア / メガサーナイト** — STAB等倍(みず/フェアリー)で先手
